# Arakanoid (Arkanoid-style Game in Java)

An Arkanoid-style game implemented in **Java** (IntelliJ project).  
The game includes a paddle, **three balls**, and **six color-coded rows** of breakable blocks, with real-time movement and collision handling.

## Features
- Paddle movement and ball control
- 3 balls running simultaneously
- 6 rows of breakable blocks (color-coded)
- Real-time game loop (update + render)
- Collision detection (balls vs. walls / paddle / blocks)

## Tech
- Java
- IntelliJ IDEA project structure
- Uses `biuoop-1.4.jar` (included in the repo)

## Repository Structure
- `src/` – source code (main implementation)
- `biuoop-1.4.jar` – external library dependency
- `biuoop.xml` – library/config resource (as provided in the project)
